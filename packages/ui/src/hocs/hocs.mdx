# High order components

## withAutoFocus
Добавляет фокус при инициализации компонента.

### Использование
```tsx
import React from 'react';
import { withAutoFocus } from '@salutejs/ui/hocs';

const Button: React.FC = (props) => <Button {...props} />
const AutoFocusButton = withAutoFocus(Button);
```
